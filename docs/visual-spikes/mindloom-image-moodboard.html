Финальные выводы — что копировать в Mindloom

Top 3 — Graph направления

🥇

Radial Organic Bubble (Kumu + Flourish + Mindloom preview)

Статичный radial layout: центральный coral bubble (28px) + 6 satellite (20px) по кругу равномерно. Gradient fill per type. Glow на central. 5 типов рёбер через цвет и dash. Нет overlap, нет physics, нет новых deps.

🥈

Semantic Cluster Map (InfraNodus aesthetic)

Кластерные halos по психологическому типу. Glow-nodes (уже частично есть). Subtle background wash на кластер. Пульсирующий central node.

🥉

Directed Edge Network (Flourish Directional style)

Thin clean arrows. No decoration. Clear arrowheads. Можно улучшить текущий SVG renderer — только поменять стиль стрелок и добавить radial layout.

Что реально копировать прямо сейчас

  • Radial layout: angle = 2π × i / n, radius=constant
  • Node gradient: radial-gradient(circle at 35% 35%, lighter, darker)
  • Edge colors: coral=hard, violet=normal, amber=soft dashed
  • Glow filter: feGaussianBlur stdDeviation="2" уже в коде
  • Hover: scale(1.07) на node SVG element

Top 3 — Speech Cloud направления

🥇

SVG Organic Blob + HTML Chips (Blobmaker style)

Static SVG blob path (warm ivory fill) как background. Поверх — flex-wrapped HTML chips. В центре blob — central phrase крупным шрифтом. Нет deps, нет JS.

🥈

Card-Cloud + Emotional Bubble (HealBubble style)

Карточка с CSS bumps сверху. Большой soft circle с central phrase. Breathing animation (scale keyframe). Chips под circle. Mobile-first, максимально надёжно.

🥉

Semantic Bubble Chart (size = significance)

Каждая phrase = bubble разного размера. Gradient fill per type. Floating layout. Самый богатый семантически.

Что реально копировать прямо сейчас

  • CSS cloud card: border-radius: 50% 45% 55% 48% / 46% 52% 48% 54%
  • Bumps: absolute div circles над карточкой
  • Breathing: @keyframes breathe { scale(1) ↔ scale(1.04) }
  • Chip semantic colors: red=heavy, amber=avoidance, violet=belief, green=resource
  • Center phrase: 15–17px, font-weight 740, text-align center
