Рекомендация
Graph — рекомендуем @xyflow/react
React Flow v12 — оптимальный выбор: нативный SSR в v12, read-only через nodesDraggable=false, полностью кастомные React-компоненты для узлов (любой дизайн), 5 типов рёбер через custom edges, mobile-friendly при отключённом pan/zoom. Bundle ~140 kB gzip — единственный серьёзный минус.

Альтернатива: улучшить текущий Pure SVG, добавив алгоритм radial layout с collision detection. Это 0 kB зависимостей и минимальный риск — но ограниченная масштабируемость.
Установка (НЕ запускать до одобрения)
npm install @xyflow/react

Потребуется: импорт CSS @xyflow/react/dist/style.css или base CSS, dynamic import с ssr: false для полной безопасности SSR.
Speech Cloud — рекомендуем Вариант 3 (Card-Cloud Hybrid)
Вариант 3 — самый надёжный, адаптивный и безопасный. Нет зависимостей. Работает на 390px mobile. Легко поддерживать. Варианты 1 и 2 более "облачные" визуально, но менее надёжны при разном кол-ве chips.
Стратегия внедрения
1. Speech Cloud — нет новых зависимостей, быстро: Patch 4P-implement-speech-cloud
2. React Flow graph — новая зависимость, отдельно: Patch 4Q-implement-react-flow-causal-graph
3. Feature flags: USE_SPEECH_CLOUD=false / USE_REACT_FLOW=false по умолчанию.
4. Rollback: старые компоненты остаются как fallback.
