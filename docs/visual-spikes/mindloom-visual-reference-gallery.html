1. Soft Bubble Causal Graph

Органические пузыри, тёплые цвета, мягкие связи — живая карта отношений 6 refs
REF-01 Kumu.io Stakeholder Map
kumu.io

Круглые "пузырьки" разного размера, force-directed layout, цветные группы. Ощущение живой карты территории, а не схемы архитектуры.

Взять для Mindloom Размер bubble = интенсивность узла. Мягкая группировка по цвету. "Карта", не "диаграмма".
🟡 Риск: средний
REF-02 Whimsical Mind Maps
whimsical.com/mind-maps

Чистые rounded-rectangle карточки. Мягкие изогнутые линии без стрелок. Цветные бордеры для различения типов. Очень readable на mobile.

Взять для Mindloom Скруглённые карточки-узлы. Кривые без стрелок → ощущение "мысли". Иерархия через размер и цвет.
🟢 Риск: низкий
REF-03 Coggle Radial Mind Map
coggle.it/gallery

Радиальная структура с органическими кривыми ветвями "как реки". Яркие soft цвета — coral, violet, amber. Текст живёт на ветке, не в рамке. Живое и поэтическое.

Взять для Mindloom Органические кривые edges. Цветовые семейства веток. Ощущение "расходящейся системы".
🟢 Риск: низкий

figma.com/templates/causal-loop-diagram

Мягкий off-white фон. Rounded rect узлы с читаемой типографикой. Curved arrows с метками типов (+/-). Объяснительный, образовательный стиль.

Взять для Mindloom Подписи типов связей. Rounded nodes с явным label. Compact legend с примерами.
🟢 Риск: низкий

Финальные рекомендации

TOP-3 направления для Graph

🥇

Radial Organic Bubble Map

Coggle + Kumu hybrid. Статичный radial layout: центральный coral bubble + 6–7 satellite пузырей по кругу. Gradient fill по типу узла. Органические кривые edges с цветом и dash = тип связи. Subtle glow на central node. Решает overlap, нет новых зависимостей.

🥈

Card-Node Explanatory Map

Heptabase + FigJam hybrid. Nodes = mini-карточки: тип (eyebrow) + label (bold) + intensity bar. Curved arrows с цветом. Click → detail sheet. Больше информации без клика.

🥉

Neural Cluster Map

InfraNodus aesthetic. Glow-nodes с halo по психологическому типу. Subtle background wash по кластерам. Пульсирующий central node. Самое "психологическое" ощущение, но риск выглядеть медицинским.

TOP-3 направления для Speech Cloud

🥇

SVG Organic Blob + HTML Chips

Blobmaker blob как background, chips поверх, central phrase в центре. Breathing animation на blob. Нет зависимостей, наиболее "облачное" ощущение.

🥈

Card-Cloud Hybrid + Emotion Bubble

Карточка с bumps сверху. Большой soft circle в centre с central phrase + дыхание. Chips вокруг. Максимально надёжно на mobile, нет зависимостей.

🥉

Semantic Bubble Chart

Каждая phrase = bubble по значимости. Floating layout. Цвет = sentiment. Самый семантически богатый, но сложнее в реализации.

Что выбрать первым

Graph: Radial Organic Bubble Map — решает реальную проблему overlap, нет новых deps, близко к текущему коду, визуально сильнее всего.
Speech Cloud: Card-Cloud Hybrid с Emotion Bubble center — самое надёжное, нет deps, mobile-first, легко как отдельный компонент.

Что НЕ выбирать

Dark InfraNodus theme Не совпадает с warm ivory Mindloom palette
Force-directed physics Нестабильность при 5–8 узлах, "прыгает"
True wordcloud algorithm Не тот UX, что нужен для speech patterns
Technical graph aesthetic Neo4j/engineering look — не психологический
3D graph visualization Overkill, accessibility проблемы, mobile риск
